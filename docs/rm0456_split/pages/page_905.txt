Bits 31:2 Reserved, must be kept at reset value.
Bit 1 FI: Force invalidate
When set, the cache entries are invalidated. This bit is reset by hardware when the
invalidation is complete. Write 0 has no effect.
0: Invalidation process complete
1: Force invalidation/invalidation process on going
Bit 0 FF: Force flush
When set, the cache entries are flushed. This bit is reset by hardware when the flushing is
complete. Write 0 has no effect.
0: Flushing process complete
1: Force flush/flushing process on going
