GFXMMU interrupts
An interrupt can be produced on the following events:
•

Buffer 0 overflow

•

Buffer 1 overflow

•

Buffer 2 overflow

•

Buffer 3 overflow

•

AHB master error

Separate interrupt enable bits are available for flexibility.
