Embedded flash memory (FLASH)

7.9.2

FLASH nonsecure key register (FLASH_NSKEYR)
Address offset: 0x08
Reset value: 0x0000 0000
Access: one wait state; word access
This register is nonsecure. It can be read and written by both secure and nonsecure access.
This register can be protected against unprivileged access when NSPRIV = 1
in FLASH_PRIVCFGR register.

7.9.3

FLASH secure key register (FLASH_SECKEYR)
Address offset: 0x0C
Reset value: 0x0000 0000
Access: one wait state; word access
This register is secure. It can be read and written only by secure access. A nonsecure
read/write access is RAZ/WI.
This register can be protected against unprivileged access when SPRIV = 1
in FLASH_PRIVCFGR register.
