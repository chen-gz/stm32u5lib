BSMX programming sequence example
The BSSEL[4:0] bitfield cannot be changed if the DFLT0 is enabled. The following steps are
needed to change the value of BSMX:
1.

40.4.7

Set DFLTEN of DFLT0 to 0.

2.

Wait for BSMXACTIVE = 0.

3.

Program BSSEL[4:0].

4.

Set DFLTEN of DFLT0 to 1.

Digital filter processing (DFLT)
The digital filter processing includes the following sub-blocks:
•

symbol remap (SBR)

•

clock skipper delay (DLY)

•

MCIC decimation filter that can be configured in Sinc4 or Sinc5

•

gain control (SCALE)

•

signal saturation (SAT)

•

reshape filter (RSFLT)

•

high-pass filter (HPF)

•

receive RXFIFO

Figure 367 shows the filter path configuration according to CICMOD[2:0]. Several
configuration bits are available to configure the digital filter to the application needs.
