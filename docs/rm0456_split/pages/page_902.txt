Bits 31:23 Reserved, must be kept at reset value.
Bits 22:21 ACLB[1:0]: Address cache lock buffer
This field selects the virtual buffer on which the address cache is active.
00: Address cache locked on buffer 0
01: Address cache locked on buffer 1
10: Address cache locked on buffer 2
11: Address cache locked on buffer 3
Bit 20 ACE: Address cache enable
This bit enables the GFXMMU address cache.
0: Address cache is disabled
1: Address cache is enabled
Bits 19:18 Reserved, must be kept at reset value.
Bit 17 OB: Outter bufferability
This bit configure the bufferability of an access generated by the GFXMMU cache.
0: No bufferable
1: Bufferable
Bit 16 OC: Outter cachability
This bit configure the cachability of an access generated by the GFXMMU cache.
0: No cachable
1: Cachable
Bits 15:13 Reserved, must be kept at reset value.
Bit 12 PD: Prefetch disable
This bit disables the prefetch of the cache.
0: Prefetch enable
1: Prefetch disable
Bit 11 FC: Force caching
This bit force the caching into the cache regardless of the MPU attributes.
The cache must be enable (CE bit set).
0: Caching not forced
1: Caching forced
