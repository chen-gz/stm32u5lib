Bits 31:21 Reserved, must be kept at reset value.
Bits 20:16 CSBOUND[4:0]: NCS boundary
This bitfield enables the transaction boundary feature. When active, a minimum value of 3 is
recommended. The NCS is released on each boundary of 2CSBOUND bytes.
0: NCS boundary disabled
Others: NCS boundary set to 2CSBOUND bytes
Bits 15:8 Reserved, must be kept at reset value.
Bits 7:0 MAXTRAN[7:0]: Maximum transfer
This bitfield enables the communication regulation feature. The NCS is released every
MAXTRAN+1 clock cycles when the other OCTOSPI request the access to the bus.
0: Maximum communication disabled
Others: Maximum communication is set to MAXTRAN + 1 bytes.
