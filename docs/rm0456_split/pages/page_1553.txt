Note:

When an activated filter did not receive a data strobe, the filter is frozen. If the interface
providing the stream is not enabled (SITF or ADCITF), the filter is frozen. Once a DFLT x is
activated, CIC, HPF, IIR and INT are reinitialized. For each filter a status bit indicates if the
filter is currently running or not.

Starting several filters synchronously
To start the acquisition of several filters synchronously, the following sequence must be
performed (assuming that DFLTEN is set to 0):
1.

Enable the CKGEN and, if needed, enable the MDF_CCK0 and MDF_CCK1 clocks.

2.

Set the SITFEN bit of the requested data interfaces to 1.

3.

For each filter, set the acquisition mode to synchronous (ACQMOD[2:0] = 01x).

4.

For each filter, set TRGSRC[3:0] to 0 (TRGO is selected).

5.

For each filter, set TRGSENS to 0 (rising edge).

6.

For each filter, set DFLTEN to 1.

7.

Read TRGO bit until it is read to 0.

8.

Set TRGO to 1. Then the acquisition sequence for all selected filters starts immediately.
