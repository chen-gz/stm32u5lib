Audio digital filter (ADF)
The TRGSENS bit allows the selection of the trigger edge (rising or falling). The trigger is
ignored if an acquisition is ongoing or if DFLTEN is set to 0.
Figure 374 shows a simplified example where the trigger logic is sensitive to a rising edge
trigger (TRGSENS = 0). The first rising edge of the trigger signal is ignored because
DFLTEN = 0. Then the next rising edge is taken into account and starts the acquisition. All
other rising edges are ignored. The trigger logic is re-initialized when DFLTRUN goes back
to 0.

The acquisition can be stopped by setting DFLTEN back to 0. This resets the filter and
