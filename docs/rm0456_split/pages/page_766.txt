LPDMA requests
A LPDMA request from a peripheral can be assigned to a LPDMA channel x, via the
REQSEL[4:0] field in the LPDMA channel x transfer register 2 (LPDMA_CxTR2), provided
that SWREQ = 0.

LPDMA block requests
Some LPDMA requests must be programmed as a block request, and not as a single
request. Then the BREQ bit in LPDMA channel x transfer register 2 (LPDMA_CxTR2) must
be set for a correct LPDMA execution of the requested peripheral transfer at the hardware
level.
