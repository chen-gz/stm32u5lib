Status flags are available in order to inform the application that an OLD event is detected.
Latched events are cleared by writing 1 into the OLDF bit.
As shown in Figure 339, when THINB = 0, the interrupt signal remains active as long as the
signal is outside the gray area. At position A, the application clears the interrupt, but the
interrupt is re-asserted because the signal is still outside the gray area. When the
application clears OLDF, THHF and THLF are cleared as well.
An OLD event can be assigned to one or several break output signals (mdf_break[3:0]). The
break signal assignment to a given OLD event is done by BKOLD[3:0] in the MDF out-of
limit detector control register x (MDF_OLDxCR).
Note:

The generation of break signals is independent from the interrupts generation.
