This bit is set and cleared by software.
0: Sound-level-ready interrupt disabled
1: Sound-level-ready interrupt enabled

This bit is set and cleared by software.

1: Sound-trigger interrupt enabled

This bit is set and cleared by software.

1: Reshape filter overrun interrupt enabled
