Bits 31:0 BKP[31:0]:
The application can write or read data to and from these registers.
In the default (ERASE) configuration this register is reset on a tamper detection event. It is
forced to reset value as long as there is at least one internal or external tamper flag being
set. This register is also reset when the readout protection (RDP) is disabled.
