RM0456

Low-power direct memory access controller (LPDMA)

18.8.2

LPDMA privileged configuration register (LPDMA_PRIVCFGR)
Address offset: 0x004
Reset value: 0x0000 0000
A write access to this register must be privileged. A read access can be privileged or
unprivileged, secure or nonsecure.
A write access is ignored at bit level if the corresponding channel x is locked
(GPDMA_RCFGLOCKR.LOCKx = 1).
This register can mix secure and nonsecure information. If a channel x is configured as
secure (LPDMA_SECCFGR.SECx = 1), the PRIVx bit can be written only by a secure (and
privileged) agent.
This register must be written when LPDMA_CxCR.EN = 0.
This register is read-only when LPDMA_CxCR.EN = 1.
This register must be programmed at a bit level, at the initialization/closure of a LPDMA
channel (LPDMA_CxCR.EN = 0), to individually allocate any channel x to the privileged or
unprivileged world.
