DSI Host (DSI)
This section applies to STM32U599/5A9/5F9/5G9 devices only.

44.1

Introduction
Portions Copyright (c) Synopsys, Inc. All rights reserved. Used with permission.
The display serial interface (DSI) is part of a group of communication protocols defined by
the MIPI® Alliance.
The MIPI® DSI Host controller is a digital core that implements all protocol functions defined
in the MIPI® DSI specification. It provides an interface between the system and the MIPI®
D-PHY, allowing the user to communicate with a DSI-compliant display.
