59.3.5

Absolute timers
Absolute time
The absolute time 32-bit value can be read directly in GFXTIM_ATR (absolute time register)
to provide a global time and to generate absolute time accurate event to synchronize the
graphical software.
The absolute time is generated by the combination of:
•

the absolute frame counter

•

the absolute line counter

Absolute frame counter
The absolute frame counter is a 20-bit free running up-counter with a software controlled
reset.
It can be enabled by writing 1 in AFCEN (absolute-frame counter enable) of GFXTIM_TCR
(timers configuration register).
