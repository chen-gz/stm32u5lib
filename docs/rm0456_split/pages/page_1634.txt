Threshold computation
The way the threshold value is computed depends on SADMOD[1:0]:
•

If SADMOD[1:0] = 0, THRH is obtained by multiplying the current ANLVL value with the
gain defined in SNTHR[3:0].
THR H = ANLVL × 10

GdB SNTHR
---------------------------20

This threshold value is then compared to the current sound level (SDLVL).
•

If SADMOD[1:0] = 01, THRH is obtained by multiplying the current ANMIN[12:0] with
the gain defined by SNTHR[3:0].
THR H = ANMIN × 10

GdB SNTHR
---------------------------20

This threshold value is then compared to the current sound level (SDLVL).
